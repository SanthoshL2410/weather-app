## Project Description

This project is a Weather App developed using Python. It allows users to check real-time weather information for different cities through a simple Command Line Interface (CLI). The application fetches weather data using an API and displays details such as temperature, humidity, weather condition, and wind speed.

The program continuously asks users to enter city names until they choose to exit the application.

## Features

* Weather Information

  * Current temperature
  * Humidity level
  * Weather condition
  * Wind speed

* City Search

  * Search weather by city name
  * View instant weather details

* Continuous Search

  * Enter multiple city names
  * Type `exit` to close the application

* Command Line Interface (CLI)

  * Simple and easy-to-use interface

## How to Run

1. Open the project folder in VS Code or IDLE
2. Install the required library:

```bash
pip install requests
```

3. Open the Python file
4. Run the program:

```bash
python weather_app.py
```

5. Enter the city name to check weather details
6. Type `exit` to close the application
